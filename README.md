How to Run Locally
1. Clone the repo
git clone https://github.com/Maandla/Tati-Full-Stack-Developer-Technical-Assessment.git
cd Tati-Full-Stack-Developer-Technical-Assessment
2. Install dependencies
pip install -r requirements.txt
3. Start the application
py run.py
The service will start on:
http://127.0.0.1:8000/meter/confirm/

Testing the Service
Each test case is included in sample_request/ and tests/. or use test_app.py

TEST CASES (REQUIRED BY THE ASSESSMENT)
1️⃣ msno="01234567890" — Successful Response
Curl
curl -X POST http://127.0.0.1:8000/meter/confirm/ \
  -H "Content-Type: text/xml" \
  --data-binary @request_examples/request_01234567890.xml
Expected Response
A valid SOAP response returning meter details (11-digit valid meter number).
Matches the exact structure in the assessment.

Tech Stack
•	Python 3
•	Flask (minimal web layer)
•	lxml (XML parsing)
•	gunicorn (production server on Render)
No frameworks outside what the assessment allows.
No autogenerated SOAP libraries — everything is manually built to match the spec.
